# APExpress — Вебресурс із продажу автозапчастин з інтегрованим AI-асистентом

**APExpress** — це сучасний e-commerce вебресурс для торгівлі автомобільними запчастинами, ключовою особливістю якого є вбудований інтелектуальний консультант на базі архітектури **RAG (Retrieval-Augmented Generation)**. Асистент дозволяє користувачам підбирати деталі та перевіряти їх сумісність за допомогою діалогу природною мовою, спираючись на внутрішню базу знань проєкту.

Проєкт виконано в межах кваліфікаційної роботи бакалавра за спеціальністю «122 Комп'ютерні науки».

---

## Ключовий функціонал

- **Інтелектуальний підбір (RAG):** Автономна підтримка клієнтів, генерація точних рекомендацій щодо сумісності автозапчастин на основі інтеграції з OpenAI API та векторного пошуку в БД.
- **E-commerce модулі:** Повноцінний каталог товарів, фільтрація, управління кошиком та оформлення замовлень.
- **Безпека:** Захищений обмін даними між клієнтом та сервером за допомогою JWT-авторизації.
- **Тестування:** Комплексна перевірка надійності системи, що включає функціональне тестування (31 ручний сценарій) та автоматизоване модульне тестування (pytest для бекенду, Jest для фронтенду).

---

## Стек технологій

- **Бекенд (Server-side):** Python, Django REST Framework (DRF)
- **Фронтенд (Client-side):** JavaScript, React.js
- **База даних:** PostgreSQL (з підтримкою векторного розширення для RAG)
- **Штучний інтелект:** OpenAI API (моделі штучного інтелекту)
- **Тестування:** pytest, coverage (Python); Jest (JavaScript)

## Інструкція із розгортання

### 1. Налаштування серверної частини (Backend)
Перейдіть до папки бекенду, налаштуйте віртуальне середовище та встановіть залежності:
```bash
# Створення та активація віртуального середовища
python -m venv myenv
source myenv/bin/activate  # Для Linux/macOS
myenv\Scripts\activate     # Для Windows

# Встановлення пакетів
pip install -r requirements.txt
```

Створіть файл `.env` у корені бекенду та додайте необхідні змінні середовища (конфігурація БД, секретний ключ Django, OpenAI API Key):
```env
SECRET_KEY=your_django_secret_key
DB_NAME=your_db_name
DB_USER=your_db_user
DB_PASSWORD=your_db_password
OPENAI_API_KEY=your_openai_api_key
```

Виконайте міграції та запустіть сервер розробки:
```bash
python manage.py migrate
python manage.py runserver
```

### 2. Налаштування клієнтської частини (Frontend)
Відкрийте новий термінал, перейдіть до папки фронтенду, встановіть залежності npm та запустіть локальний сервер:
```bash
cd frontend
npm install
npm start
```

## Запуск тестів

- **Бекенд тести (pytest):**
  ```bash
  pytest
  ```
- **Збір метрик покриття коду (coverage):**
  ```bash
  coverage run -m pytest
  coverage report
  ```
- **Фронтенд тести (Jest):**
  ```bash
  cd frontend
  npm test
  ```
